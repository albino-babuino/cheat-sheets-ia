# 11 - Deep Learning

Directorio dedicado a **cheat sheets** de Deep Learning (aprendizaje profundo).

Aquí puedes añadir notebooks sobre:

- Redes neuronales profundas (DNN)
- Frameworks: TensorFlow/Keras, PyTorch
- CNN (redes convolucionales)
- RNN, LSTM, GRU
- Transfer learning y fine-tuning
- Optimizadores, regularización, callbacks
- Vision, NLP, series temporales con redes profundas

Los contenidos de IA moderna (redes básicas, TensorFlow, CNN, RNN) también están en **10 - IA Moderna**; este directorio sirve para ampliar y profundizar en temas específicos de Deep Learning.
